Easy Swift Script Reloading

I was working on a Swift script that needed to stay running, so I added a loop to avoid interruptions. Always on!

My solution used a while loop with Process instances to re-run the code. Script inception!

I realized this setup could be used as a basic hot-reloading tool for script development.

let refresh = "refresh" if Process.arguments.contains(refresh) { while true { let task = Process() task.executableURL = URL(fileURLWithPath: Process.arguments[0]) task.arguments = Process.arguments.dropFirst().filter { $0 != refresh } try? task.run() task.waitUntilExit() } } Add this code to the top of your script; it loops if called with the refresh argument.

You can then launch it from the terminal:

main.swift refresh more arguments Once active, edit and save; the next run includes changes. Compile errors? No problem, the prior code stays in memory and retries.

It's not perfect and console output may be noisy, but I've found it handy for iterative development of small scripts.

Remember to use signal handlers to properly terminate the script when necessary.
